1. Opportunity Overview

Problem Statement

MedStar's medical IoT devices (e.g., insulin pumps, PACS workstations) present significant security vulnerabilities:

  • ⚠️Unpatched vulnerabilities: CVE-2023-24955 identified in 60% of devices per recent security scan
  • 🔓Default credentials: Many devices still using factory settings (admin:password)
  • 🌐No network segmentation: Devices exposed to lateral movement attacks

Proposed Solution

On-Premises IoT Hardening Program featuring:

  • 🔍Automated vulnerability scanning (no cloud dependency required)
  • 🛡️Zero-trust network segmentation via VLANs
  • 🔧Custom patching solutions for legacy medical devices
